Удобное масштабирование с REM

Публикация: 16-01-2018 | Теги: tmux

При правильном использовании rem станет хорошим помощником в создании адаптивного веб-сайта.

1 rem всегда равен font-size, заданному html

Чтобы не удобно расчитывать размер шрифта в rem, лучше изначально прописать его размером в 10px.

html{
  font-size: 10px
}

После этого все единицы измерения в CSS с px нужно переписать на rem. На пример шрифт размером в 18px переписать размером в 1.8rem, отступ в 5px станет равен 0.5rem и так далее.

Когда все размеры веб-сайта буду заданы в rem, масштабирование всех элементов на сайте можно будет изменять с помощью изменения размера 1 rem.

Например, если при разрешении меньше 1440px нужно уменьшить размеры веб-сайта на 20%, тогда можно использовать следующий код.

@media all and (max-width: 1440px){
  html{
    font-size: 8px;
  }
}

Используя подобные конструкции можно удобно масштабировать веб-сайт, согласно разрешению экрана пользователя.
