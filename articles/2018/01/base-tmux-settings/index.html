Rem means responsible

Публикация: 16-01-2018 | Теги: tmux

При правильном использовании rem станет хорошим помошником в создании адаптивного веб-сайта.

Для удобного использования rem, если под рукой нет калькулятора, а самому напрягаться лень- лучше изначальный прописать размер 1rem равного 10px.

html{
  font-size: 10px
}

После этого все единицы измерения в css с пикселей нужно перписать на rem. Например шрифт размером в 18px переписать размером в 1.8rem, 16px в 1.6 rem и т.д.

Если все размеры веб-сайта буду переписаны, то можно использовать главное свойство rem - изменить размер одного рема, что повлечет за собой изменения размеров элементов, где в размерах указаны ремы, на всем сайте.

Например при разрешении меньше 1440px нужно сделать все элементы веб-сайта меньше на 20%, тогда можно использовать следующий код.

@media all and (max-width: 1440px){
  html{
    font-size: 8px;
  }
}

Используя такую конструкцию, можно изменять размеры всех элементов веб-сайта парой строчкек кода.
