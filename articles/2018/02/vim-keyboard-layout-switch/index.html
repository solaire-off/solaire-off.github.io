Лучший способ смены раскладки в Vim

Публикация: 14-02-2018 | Теги: #vim

После перехода на работу Vim встал вопрос — как сменить раскладку в режиме ввода, чтобы при выходе в нормальный режим не приходилось сменять раскладку назад на латиницу, ведь горячие клавиши работают только на ней.

Эта головная боль тех пользователей, чей язык не использует латинский алфавит.

Способы решения разнятся от использования плагинов в Vim до настройки системных утилит, некоторые решения страдают привязкой к определённой ОС, другие сложностью в их исполнении.

Лучшее же решение этой проблемы заложено в базовый функционал самого Vim и настраивается парой строчек в vim.rc.

Сначала надо указать Vim какие раскладки будут использоваться, для ru/en прописываем:

set keymap=russian-jcukenwin

Теперь переключение раскладки в режиме ввода доступно по стандартной комбинации <C-^>, что само по себе неудобно, но уже позволяет не терять время на возвращение раскладки латинский вариант после выхода в «нормальный» режим.

Для себя выбрал выбрал <C-L> в качестве комфортной комбинации смены раскладки.

# Для режима ввода
imap <C-L> <C-^>
# Для коммадного режима
cmap <silent> <C-L> <C-^>

Для переключения раскладки в «нормальном» режиме, например для того чтобы такие команды f и F искали символы на нужном языке, я использую сочетание клавиш <Leader>L, так как в моем конфиге в «нормальном» режиме команда <C-L> уже используется.

nmap <silent> <Leader>l a<C-^><Esc>
