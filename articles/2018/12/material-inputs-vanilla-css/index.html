Поле ввода в стиле материал на чистом CSS

Публикация: 15-12-2018 | Теги: css

Множество форм в стиле от Google используют Java Script для отслеживания своего состояния и реализации анимации элементов, и все бы хорошо, но иногда возникает необходимость сверстать такую форму без использования скриптов, рассмотрим как такое реализовать, использую только CSS.

Начнем с html разметки.

<div class="form-control">
  <input id="id_login" class="form-control__input" name="login" placeholder=" ">
  <label for="id_login" class="form-control__label">Entel login</label>
</div>

Добавим стилей, где стилизуем label в виде placeholder.

.form-control
  position: relative  
  &__label
    position: absolute
    left: 0
    top: 0
    cursor: default
    display: block
    color: gray
  &__input
    padding: 0 0 2px 0
    border: none
    border-bottom: 2px solid rgba(gray, 0.7)

К реализации анимации: по фокусу на поле добавим раскрашивания границы и переместим label вверх, для этого воспользуемся псевдоклассом focus.

.form-control
  position: relative  
  &__label
    position: absolute
    left: 0
    top: 0
    cursor: default
    display: block
    color: gray 
    transition: all 0.2s
  &__input
    padding: 0 0 2px 0
    border: none
    border-bottom: 2px solid rgba(gray, 0.7)
    &:focus
      border-color: #ffdb4d    
      + .form-control__label
        top: -1em
        font-size: 80%

Для того, чтобы сохранить положение label, когда пользователь закончит ввод, воспользуемся псевдоклассом placeholder-shown. Следует отметить, что для работы псевдокласса необходимо наличие самого placeholder, только в нашем случае он был пустым placeholder=" ".

.form-control
  position: relative  
  &__label
    position: absolute
    left: 0
    top: 0
    cursor: default
    display: block
    color: gray 
    transition: all 0.2s
  &__input
    padding: 0 0 2px 0
    border: none
    border-bottom: 2px solid rgba(gray, 0.7)
    &:focus
      border-color: #ffdb4d    
      + .form-control__label
        top: -1em
        font-size: 80%
    &:not(:placeholder-shown)
      + .form-control__label
        top: -1em
        font-size: 80%
