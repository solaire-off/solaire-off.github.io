Масштабирование веб-сайта с помощью REM

Публикация: 16-12-2018 | Теги: #css

Когда в мобильной версии веб-сайта необходимо уменьшить размеры компонентов, на помощь может прийти такая единицы измерения как rem. При правильном использовании rem станет хорошим помощником в создании адаптивного веб-сайта.

Для начала работы с rem необходимо задать ему размер, 1rem всегда равен размеру шрифта html.

Чтобы было удобно рассчитывать размеры элементов, стоит прописать размер одного rem в 10px.

html{
  font-size: 10px
}

После этого все размеры компонентов в CSS нужно описывать в rem. Например шрифт размером в 18px станет размером в 1.8rem, отступ в 5px превратиться 0.5rem и так далее.

Когда все размеры веб-сайта буду заданы в rem, изменить масштаб всех элементов на сайте можно, изменяя количество пикселей в одном rem.

Например, если при разрешении меньше 1440px нужно уменьшить элементы веб-сайта на 20%, тогда можно использовать следующий код.

@media all and (max-width: 1440px){
  html{
    font-size: 8px;
  }
}
